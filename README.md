# Фильтрация авиа перелётов

Решение тестового задания для позиции Junior Java Developer. Модуль фильтрации перелётов по динамическим правилам.

## Архитектура

- **Паттерн Стратегия**: интерфейс `FlightFilter` + независимые реализации
- **Иммутабельность**: фильтрация не модифицирует исходные данные
- **Расширяемость**: новые правила добавляются без изменения существующего кода

### Фильтры
| Правило | Класс |
|---------|-------|
| Вылет не в прошлом | `DepartureBeforeNowFilter` |
| Валидность сегментов | `InvalidSegmentsFilter` |
| Время на земле ≤ 2ч | `LongGroundTimeFilter` |
